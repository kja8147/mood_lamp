# mood_lamp
무드등 램프 어플

## 개발시기 
  * 2018년 

## 개요
  * 무드등과 해당 기기와 연결된 어플로 구성된 프로젝트.
  * 무드등은 라즈베리 파이로 만들어 졌으며, 어플은 안드로이드 스튜디오로 만들었다.
  * 두 기기간의 연결은 블루투스로 이루어 짐.
  
## 개발 도구
  * Java
  * Android Studio
  * Rasberry Pi
  
## 기능
  * 램프 : 조명 전환
  * 어플 : 조명색상 선택, 보이스 기능(램프끄기, 램프켜기)
